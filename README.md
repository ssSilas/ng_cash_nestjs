# Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
#Install dependencies
$ npm install
```

## Running the app

```bash
# development watch mode
$ npm run dev
# debug mode
$ npm run debug

```
